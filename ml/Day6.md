## 6일차 - Tensorflow(JS) : 오리엔테이션 ~ 남의 모델 이용하기 (2021-01-11)

### 지도학습의 작업순서

지도학습의 과정을 정리 해 보자,,

1. 과거의 데이터를 준비합니다
2. 모델의 모양을 만듭니다
3. 데이터로 모델을 학습(FIT)합니다
4. 모델을 이용합니다

이 과정을 가지고 있어야 도구 변화에 영향을 받지 않게 됨!

### 수업의 순서

Tensorflow.js에서는 모델의 이용 방법을 세가지로 분류함!

- 기존에 존재하는 모델 실행
- 기존에 존재하는 모델 재학습(수정)
- 직접 개발 ('지도학습의 작업순서' 참고)
